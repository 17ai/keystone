Transformer

abstract class Transformer[A, B] extends Serializable

Transformers are functions that may be applied both to single input items and to RDDs of input items. They may be chained together, along with Estimators and LabelEstimators, to produce complex pipelines.

A

input item type the transformer takes

B

output item type the transformer produces
