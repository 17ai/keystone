GMMFisherVectorEstimator

case class GMMFisherVectorEstimator(k: Int) extends OptimizableEstimator[DenseMatrix[Float], DenseMatrix[Float]] with Product with Serializable

Trains either a scala or an enceval Fisher Vector implementation, via estimating a GMM by treating each column of the inputs as a separate DenseVector input to GaussianMixtureModelEstimator

Automatically decides which implementation to use when node-level optimization is enabled.

k

Number of centers to estimate.
