BatchPCATransformer

case class BatchPCATransformer(pcaMat: DenseMatrix[Float]) extends Transformer[DenseMatrix[Float], DenseMatrix[Float]] with Logging with Product with Serializable

Performs dimensionality reduction on an input dataset where each input item is an NxD array and the projection matrix is a DxK array.

pcaMat

A DxK projection matrix.
